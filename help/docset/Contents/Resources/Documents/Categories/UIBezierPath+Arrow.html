UIBezierPath(Arrow) Category Reference

Declared in UIBezierPath+Arrow.h

Overview

UIBezierPath+Arrow constructs a bezier path for a typical block arrow with filled head.

+ bezierPathWithArrowFromPoint:toPoint:tailWidth:headWidth:headLength:

Constructs a bezier path for an arrow. The tail and tip points define the centre of the tail and tip of the arrow, respectively, and the arrow’s path is created in between for the specific constrains.

+ (UIBezierPath *)bezierPathWithArrowFromPoint:(CGPoint)tailPoint toPoint:(CGPoint)tipPoint tailWidth:(CGFloat)tailWidth headWidth:(CGFloat)headWidth headLength:(CGFloat)headLength

Parameters

tailPoint

The point that indicates the centre of the arrow’s tail
tipPoint

The point that indicates the tip of the arrow’s head
tailWidth

Specifies the size of the tail width
headWidth

Specifies the size of the head width
headLength

Specifies the size of the head length

Return Value

The bezier path of the arrow
